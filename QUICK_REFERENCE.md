## 💡 First-Time Setup (One-Time)

1. Clone repository
2. Run `npm install`
3. Run `npm run dev`
4. Open browser to `http://localhost:5000/ipor-fusion-vault-la/`
5. Start creating vaults!

## 🔌 Wallet Setup (Optional)

2. Click "Connect Wallet" in app
3. Approve connection
4. Select network (Sepolia for testing)
5. Get test ETH: https://sepoliafaucet.com

## 🆘 Troubleshooting

**App won't load?**
- Clear cache, try different browser

**Can't connect wallet?**
- Install MetaMask, approve connection

**Deploy fails?**
- Check ETH balance, verify network
